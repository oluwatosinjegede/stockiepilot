# StockiePilot

StockiePilot is a Django-based inventory and sales management platform designed for modern retail and operations teams.

## Core capabilities
- Product and inventory tracking
- Sales creation and analytics
- Subscription and access control flows
- User and company management

## Public pages
The app includes a modern marketing homepage and legal/compliance pages:
- `/readme/`
- `/privacy-policy/`
- `/terms-of-service/`
- `/data-protection/`
- `/fraud-warning/`

## Local development
1. Create a virtual environment.
2. Install dependencies from `requirements.txt`.
3. Run migrations.
4. Start Django with `python manage.py runserver`.
